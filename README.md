# ojp-rest-api

Models part of the Open Journey Planner (OJP) API as a REST service.

Based on [VDVde/OJP](https://github.com/VDVde/OJP/tree/changes_for_v1.1) GitHub.

## Scope

Currently in scope is `OJPDelivery` of `ServiceDelivery`.
